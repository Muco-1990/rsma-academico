\section*{Questões}

\textbf{1. O que é um software de revisão sistemática?}
\begin{enumerate}[label=\alph*)]
    \item Uma ferramenta para escrever artigos sem referências
    \item Um programa para organizar, gerenciar, avaliar e sintetizar a literatura científica
    \item Um software de estatística geral
    \item Um gerenciador de emails
\end{enumerate}

\textbf{2. Qual é o principal objetivo do EPPI-Reviewer Web?}
\begin{enumerate}[label=\alph*)]
    \item Apenas escrever resumos de artigos
    \item Triagem, codificação, projetos colaborativos e meta-análise
    \item Criar apresentações de PowerPoint
    \item Gerenciar contas bancárias
\end{enumerate}

\textbf{3. Qual dos seguintes softwares é oficial da Cochrane?}
\begin{enumerate}[label=\alph*)]
    \item Rayyan
    \item JBI SUMARI
    \item RevMan
    \item Mendeley
\end{enumerate}

\textbf{4. Quais funcionalidades o RevMan oferece? (múltipla seleção)}
\begin{itemize}
    \item Organização de referências e estudos incluídos
    \item Meta-análise com gráficos (forest plots, funnel plots)
    \item Tradução automática de artigos
    \item Avaliação de risco de viés
    \item Exportação para publicação
\end{itemize}

\textbf{5. O que significa que um software de revisão sistemática garante transparência e rastreabilidade?}
\begin{enumerate}[label=\alph*)]
    \item Permite que qualquer pessoa veja todas as etapas do estudo
    \item Não permite que revisores alterem dados
    \item Apenas gera gráficos
    \item Faz download automático de artigos
\end{enumerate}

\textbf{6. Qual software é mais indicado para triagem rápida e colaboração entre revisores?}
\begin{enumerate}[label=\alph*)]
    \item RevMan
    \item Rayyan
    \item JBI SUMARI
    \item Excel
\end{enumerate}

\textbf{7. Sobre o acesso ao RevMan: qual afirmação está correta?}
\begin{enumerate}[label=\alph*)]
    \item É pago para todos os usuários
    \item É gratuito apenas para membros Cochrane; versão básica gratuita para não membros
    \item Funciona apenas online
    \item Só é usado para revisão narrativa
\end{enumerate}

\textbf{8. Quais objetivos principais um software de revisão sistemática busca atender? (múltipla seleção)}
\begin{itemize}
    \item Facilitar busca, seleção e triagem de estudos
    \item Automatizar extração de dados e avaliação de qualidade
    \item Auxiliar na síntese qualitativa e quantitativa
    \item Garantir transparência e rastreabilidade
\end{itemize}
